df = pd.read_csv("http://www.stat.wisc.edu/~jgillett/451/data/iris.csv")
df

df2 = df[df['Species'] != 'setosa']
df2 = df2[['Petal.Length', 'Species']]

y = (df['Species'] == 'virginica').to_numpy().astype(int)

X= df['Petal.Length'].to_numpy()
X = X.reshape(-1,1)

model = linear_model.LogisticRegression(C=1000)

model.fit(X, y)

b = model.intercept_
w = model.coef_[0]
estimate = model.predict_proba([[5]])[:,1]
score = model.score(X,y)
predicted = model.predict([[5]])
print(f' The accuracy on the training data is {score}.')
print(f' The Estimated probability of virginica given Petal Length = 5 is {estimate}.')
print(f' The Predicted species of Petal Length = 5 is {predicted} which is Virginica.')

1b. Do some work with logistic regression by hand.¶

Consider the logistic regression model, $P(y _i = 1) = \frac{1}{1 + e^{-(\mathbf{w x} + b)}}\,.$

Logistic regression is named after the log-odds of success, $\ln \frac{p}{1 - p}$, where $p = P(y_i = 1)$. Show that this log-odds equals $\mathbf{w x} + b$. (That is, start with $\ln \frac{p}{1 - p}$ and connect it in a series of equalities to $\mathbf{w x} + b$.)

... your Latex math in a Markdown cell here ...¶

$\begin{align*} % In this Latex context, "&" separates columns and "\\" ends a line. \ln \frac{p}{1 - p} & = \ln(p) {-} ln(1-p) \\ & = \ln \frac{1}{1+e^{-(wx+b)}} {-} \ln(1 - \frac{1}{1+e^{-(wx+b)}}) \\ & = \ln(1) {-} \ln(1+e^{-(wx+b)}) {-} \ln(\frac{1+e^{-(wx+b)}-1}{1+e^{-(wx+b)}}) \\ & = -(\ln(e^{-(wx+b)}) \\ & = \mathbf{w x} + b\\ \end{align*} $

2d. Do some decision tree calculations by hand.¶

Consider a decision tree node containing the following set of examples $S = \{(\mathbf{x}, y)\}$ where $\mathbf{x} = (x_1, x_2)$:

((4, 9), 1)

((2, 6), 0)

((5, 7), 0)

((3, 8), 1)

Find the entropy of $S$.

p1 = 2/4 = .5

p0 = 2/4 = .5

log2(p1) = -1 log2(p0) = -1

H(S) = -(.5-1 ) - (.5 -1 ) = 1

The entropy is 1 because it is perfectly mixed with two 0's and two 1's in the y-value of the pairs.
