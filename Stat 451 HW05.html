1. Algorithm selection for multiclass classification by optical recognition of handwritten digits¶

The digits dataset has 1797 labeled images of hand-written digits.

  • $X$ = digits.data has shape (1797, 64).
    • Each image $\mathbf{x}_i$ is represented as the $i$th row of 64 pixel values in the 2D digits.data array that corresponds to an 8x8 photo of a handwritten digit.
  • $y$ = digits.target has shape (1797,). Each $y_i$ is a number from 0 to 9 indicating the handwritten digit that was photographed and stored in $\mathbf{x}_i$.

topAccuracy = -np.Inf
index = -1
topClass = None 

classifiers = [svm.SVC(), linear_model.LogisticRegression(), DecisionTreeClassifier(), KNeighborsClassifier()]

SVM = {'kernel':('linear', 'rbf'), 'C':[.01, 1, 100]}
logist = {'max_iter': [5000], 'C':[.01, 1, 100]}
ID3 = {'criterion': ['entropy'], 'max_depth': [1, 3, 5, 7]}
kNN = {'n_neighbors': [1, 2, 3, 4]}
param = [SVM, logist, ID3, kNN]


for i in range(len(classifiers)):
    clf = GridSearchCV(classifiers[i], param[i])
    clf.fit(X_train, y_train)
    accuracy = clf.score(X_valid, y_valid)
    if topAccuracy == 0 or accuracy > topAccuracy:
        index = i
        topAccuracy = accuracy
        topParam = clf.best_params_ 
        topClass = clf 
    
        
print(f'Best Index: {index}, Best Classifier: {topClass}, the best Accuracy is {topAccuracy:.3}, the best parameters are {topParam}')
